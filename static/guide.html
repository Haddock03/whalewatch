Indicateurs Techniques

Lire la dynamique de marché à travers RSI, EMA, MACD et Volume — les briques fondamentales de toute analyse.

RSI EMA 20/50/200 MACD Bollinger Volume

RSI — Relative Strength Index

RSI = 100 − 100 / (1 + RS)
RS = moyenne gains / moyenne pertes (14 périodes)
RSI > 70SURACHAT
RSI 50–70Zone haussière
RSI 30–50Zone neutre/baissière
RSI < 30SURVENTE
Divergence haussière : prix fait un plus bas, RSI fait un plus haut → retournement probable. Signal fort en confluence avec un support.

EMA — Exponential Moving Average

Moyenne mobile qui pondère davantage les prix récents. Plus réactive que la SMA.

EMA 20Tendance court terme (swing)
EMA 50Tendance moyen terme
EMA 200Tendance long terme (macro)
Golden CrossHAUSSIER EMA50 > EMA200
Death CrossBAISSIER EMA50 < EMA200
Prix au-dessus de l'EMA200 = biais long dominant. En dessous = biais short ou attente.
Indicateurs supplémentaires

MACD

MACD Line = EMA12 − EMA26
Signal = EMA9 du MACD

Croisement MACD ↑ Signal = acheter
Croisement MACD ↓ Signal = vendre
Histogramme grandissant = momentum fort

Bollinger Bands

Bandes = EMA20 ± 2 déviations standards

Prix touche la bande haute = surachat possible
Prix touche la bande basse = survente possible
Squeeze (bandes resserrées) = explosion de volatilité imminente

Volume

Confirme ou infirme les mouvements de prix

Prix ↑ + Volume ↑ = tendance validée
Prix ↑ + Volume ↓ = pullback probable
Volume spike soudain = manipulation ou news

Order Flow & Liquidation Maps

Identifier où se trouvent les grandes positions et anticiper les cascades de liquidations pour ne pas se faire piéger.

Liquidation Maps

Comment ça fonctionne

  • Les traders avec levier (10x, 20x, 50x) ont un prix de liquidation forcée
  • Quand le prix atteint cette zone, la position est fermée automatiquement — créant de la pression supplémentaire
  • Si beaucoup de liquidations s'accumulent au même niveau → cascade (effet domino)
  • Les market makers et bots chassent ces zones pour déclencher les stops

Stratégie pratique

Grande zone de liquidations SHORT au-dessus → le prix sera attiré vers le haut pour les déclencher
Grande zone de liquidations LONG en dessous → risque de chute rapide
Open Interest élevé + prix range = mouvement violent imminent

ATR — Average True Range

Formule

ATR = moy. mobile(14) de True Range
TR = max(H-L, |H-Cprec|, |L-Cprec|)

L'ATR mesure l'amplitude moyenne des mouvements. Un ATR élevé = terrain de jeu pour les traders. Un ATR faible = marché endormi.

Bots : Configuration & Supervision

Automatiser le trading sans sacrifier la sécurité. Les règles d'or pour que ton bot reste ton allié et non ton ennemi.

API sécurisée Stop-loss dur Backtesting Paper trading Monitoring 24/7
Sécurité API — Règles absolues
  • Permissions : Read + Trade uniquement — jamais Withdraw
  • Whitelister l'IP du serveur du bot dans les paramètres API
  • Stocker les clés dans des variables d'environnement (.env), jamais en dur dans le code
  • Utiliser une clé séparée par bot et par exchange
  • Rotation des clés tous les 90 jours
  • Activer les alertes email pour chaque ordre passé
Ne jamais stocker les clés API sur GitHub, Discord, ou dans le code source. Un seul leak = compte vidé en secondes par des scanners automatiques.

Signaux Macro

Le cycle macro impacte profondément toute la crypto. BTC Dominance, ETFs, décisions Fed et Coinglass donnent le contexte global obligatoire.

BTC Dominance ETFs Spot Fed / DXY Funding Rate
BTC Dominance (BTC.D)

Lecture de BTC.D

BTC.D monteCapital vers BTC → alts chutent
BTC.D baisseAlt season → rotations vers alts
BTC.D > 60%Bear market alts — trader BTC/USDT
BTC.D < 45%Alt season mature — rotations risquées
Décisions Fed & Macro
  • Taux Fed ↑ (hawkish) → DXY monte → risk assets baissent → crypto vend
  • Taux Fed ↓ (dovish) → liquidités → crypto monte
  • CPI (inflation) élevé → Fed hawkish → pression baissière
  • Dates clés : FOMC, CPI, NFP — anticiper la volatilité
ETFs Bitcoin/ETH — Flux entrants/sortants

Pourquoi c'est important

Entrées ETF positives = capital institutionnel qui entre → signal haussier fort
Sorties ETF nettes pendant plusieurs jours = pression vendeuse institutionnelle
Divergence prix/flux ETF = signal de retournement imminent

Funding très positifSHORTS à privilégier

Funding négatifLONGS à privilégier

Le funding rate est payé entre longs et shorts toutes les 8h. Un funding très positif = trop de longs = squeeze probable.

Adapter la Stratégie au Cycle

Il n'existe pas de stratégie universelle. La même technique qui performe en bull market détruit le capital en bear. Adapter = survivre.

NUPL MVRV Pi Cycle Top Halving cycle

Bull Market

  • HODL + DCA surperforment le trading actif
  • Acheter les dips sur EMA50
  • Rotation progressive vers alts après BTC +50%
  • Pyramider les positions gagnantes
  • Prendre des profits réguliers en stablecoins (10–20%)

Marché Latéral

  • Range trading : acheter support, vendre résistance
  • Identifier les bornes avec précision
  • Réduire la taille des positions (volatilité faible)
  • Favoriser les paires stable/vol
  • Grid bot très efficace dans ce contexte

Bear Market

  • Cash is king — stablecoins majority
  • Shorter les rebonds sur résistances
  • DCA long terme sur BTC/ETH uniquement
  • Éviter les alts (chutent 2–10x plus que BTC)
  • Accumuler discrètement en peur extrême

Halving Cycle BTC

Bitcoin se réduit de moitié tous les ~4 ans. Historiquement :

Mois -12 (pre-halving)Accumulation — prix monte
Mois 0 (halving)Event — prix souvent stable
Mois +6 à +18Bull run principal
Mois +18 à +36Bear market — purge

Passé ≠ futur. Avec les ETFs institutionnels, les cycles peuvent être moins prononcés.
